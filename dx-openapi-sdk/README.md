项目简介：
  这是关于钱包MinPay的项目。做的是关于商户api请求sdk。商户可通过本项目的测试用例，模拟商户api请求实现数字币代收、代付的订单创建业务。

项目用法：
  1. 引入本项目依赖包。
  2. 商户配置好ApiConfig类。（此配置类包含商户的baseUrl、key、secret，代码已有注释说明）
  3. 直接启动测试用例。模拟商户数字币的代收、代付等订单创建业务。
